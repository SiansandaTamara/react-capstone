## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Web Browser
- Code Editor
- Prior knowledge of CSS, HTML, GIT, GITHUB and Linters.

### Setup


```bash
npm install
```

```bash
npm build
```

```bash
npm start
```

### Run the following commands to run linter tests

WebHint
```bash
npx hint .
```

StyleLint
```bash
npx stylelint "**/*.{css,scss}"
```

ESLint
```bash
npx eslint .
```

```bash

git clone//https://github.com/SiansandaTamara/react-capstone2
cd react-capstone2.
```
